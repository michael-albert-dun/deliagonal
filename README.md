# Candies2d
My first javascript game. Select (at most two) tiles by clicking. Hit  to clear the rectangle they bound provided they're the same colour. Try to clear the whole field in as few moves as possible (one is obvious when available, spotting twos is already a bit of a task though purely mechanical).

Much code derived from ideas at [Kenny Yip Coding](https://www.kennyyipcoding.com/) and suggestions from code copilot.
